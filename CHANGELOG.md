# Changelog

此处记载了 SUSTechThesis 中所有值得留意的改动，格式参照 [Keep a Changelog](https://keepachangelog.com/en/1.0.0/)。

点击版本号即可在 GitHub 上查看相邻版本间的代码变动。

Changelog 维护应保持最新版本在文件上面的原则，日期格式按照 ISO 8601 的标准: `YYYY-MM-DD`.

#### 变更分类：*major*, *minor*, *bug-fix*。

#### *major* release trigger:

- `Changed`：现有功能变更。
- `Removed`：现有功能移除。

#### *minor* release trigger:

- `Added`：引入新功能
- `Deprecated`：标记即将被删除的功能。

#### *bug-fix* release trigger:

- `Fixed`：错误修复.
- `Security`：安全漏洞修复.

#### 使用提示

- 未发布版本使用形如 "`[Unreleased]`" 作为二级标题，
- 预发布版本使用形如 "`[1.0.2-rc.1] - 2021-06-11`" 作为二级标题，
- 正式发布版本使用形如 "`[1.0.1] - 2020-07-25`" 作为二级标题。

## [1.3.0] - 2021-08-01

### Changed

- 【封面】封面“硕士学位论文” 修改为宋体 24 号字加粗。
- 【封面】封面论文题目中文采用黑体 22 号字。
- 【封面】封面作者及指导教师信息采用宋体 18 号字。
- 【封面】封面日期填写学位论文成文打印的日期，采用用宋体 18 号字。


- 【题名页】题目采用中文黑体 22 号字，英文Times New Roman 22号字。
- 【题名页】增加英文题名页内容表格。


- 【其他】前言处增加“如果范例中存在与本《写作指南》中的规定不符之处，以《写作指南》中的文字叙述为准。”。
- 【其他】目录中的次级标题要依次缩进 1 个汉字符。
- 【其他】正文从右页开始，(根据需要在打印时添加空白页，提交电子版不需要多余空白页)。
- 【其他】论文段落文字格式中增加“中文撰写的论文除参考文献外均使用全角圆括号，仅在参考文献著录中使用半角圆括号。”。
- 【其他】标题格式中增加“其中，摘要、目录、结论、致谢等部分的标题为两个字，两字中间空 1 个汉字符（或 2 个半角空格），如“摘　要”、“目　录”、“结　论”、“致　谢”，采用章标题格式。”，关于条目标题，增加“首行左缩进 2 个汉字符，避免悬挂缩进，”。
- 【其他】用图或表段前段后空磅数代替“图、表与正文之间要有一行的间距”的要求。
- 【其他】声明在致谢之后不需要单独一页印刷。


- 【其他】学术成果用节标题，前面不加（一）（二）等，序号从[1]开始往后递推。
- 【其他】英文摘要的标题，采用英文撰写的标题全大写用ABSTRACT，中文撰写的用Abstract。
